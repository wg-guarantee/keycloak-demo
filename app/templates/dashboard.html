📊 本次登录流程回顾

  1. 浏览器访问 /dashboard → require_login 检测到未登录
  2. 重定向到 /auth/login → authlib 生成 PKCE code_challenge
  3. 浏览器跳转到 Keycloak 登录页（含短信 OTP 验证）
  4. Keycloak 验证通过 → 回调 /auth/callback?code=xxx&state=xxx
  5. authlib 验证 state，用 code + code_verifier 向 Keycloak 换取 Token
  6. authlib 通过 JWKS 验证 ID Token 签名，解析 claims
  7. 用户信息写入 Session → 重定向回 Dashboard ✅
