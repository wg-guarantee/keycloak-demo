📋 Authorization Code Flow 流程

# 步骤 谁执行 说明
① 访问应用 用户浏览器 访问 /dashboard，未登录
② 重定向到 Keycloak 应用 (authlib) 自动生成 PKCE code_challenge，跳转到 Keycloak 登录页
③ 用户登录（含短信验证） Keycloak 输入账号密码 + 短信 OTP，Keycloak 完成验证
④ 回调 + Authorization Code Keycloak → 应用 携带 code 重定向到 /auth/callback
⑤ Code 换 Token 应用 (authlib) 后端用 code + code_verifier 换取 Access/ID/Refresh Token
⑥ Token 验证 + 存 Session authlib 自动 验证 ID Token 签名（JWKS），解析 claims，写入 Session
⑦ 访问受保护资源 用户浏览器 Session 中有用户信息，正常访问业务页面和 API
